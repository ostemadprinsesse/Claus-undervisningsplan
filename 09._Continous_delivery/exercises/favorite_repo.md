# Investigate you favorite repo

**Type**: Individual

**Motivation**: Get a hang of how workflows are written.

---

In this exercise you should find a repository on GitHub that contains a .github folder with a workflow file in it. 

Then analyse this file, and write down what happens in each step.    

An example could be this repository:

* https://github.com/streamlit/streamlit

Clone the repsitory and use Mistral-vibe to investigate the workflow files. First get an overview of the files and what each of their aims are.
