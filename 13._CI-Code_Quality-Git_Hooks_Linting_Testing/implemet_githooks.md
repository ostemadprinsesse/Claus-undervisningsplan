# Implement Git Hooks

**Type**: Group
**Motivation**: Catch errors early before commits

---

In your own projects you should implement at least Linting of your code using Git Hooks.

It is a good idea to implement a tool like `pre-commit` that fits to you language (javascript: Husky etc.).

You should also in the group agree opon what rules to enforce. 

It is also a very good idea to start creating unittests for you code base and it is a good idea to run these tests before each commit.
